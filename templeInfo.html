高雄市已裝設環保金爐寺廟一覽 空氣污染防制設備優缺點說明
已裝設環保金爐寺廟一覽
# 寺廟名稱 行政區 地址 教別 防制設備
空氣污染防制設備優缺點說明
  • 重力沉降室檔板收集器
    優點：
    1. 壓力損失低
    2. 設計及維護簡易
    缺點：
    1. 需求空間大
    2. 收集效率低(約50%)
    3. 對細微粉塵無法去除
  • 旋風集塵器
    優點：
    1. 所需空間小，收集效率約85%
    2. 收集塵粒乾，可連續處置
    3. 低至中度壓損
    4. 處理大型顆粒
    5. 不受溫度影響
    缺點：
    1. 需較高的垂直距離
    2. 收對小顆粒收集效率低
  • 濕式洗滌塔
    優點：
    1. 可同時處理氣狀污染物
    2. 冷卻並清除高溫、高濕度氣體
    3. 減低粉塵爆炸之危險
    4. 收集效率約85%
    缺點：
    1. 腐蝕性問題
    2. 增加廢水處理問題
    3. 排氣易受氣體影響而受污染
    4. 濕度過高氣候下，易產生白煙
  • 靜電集塵器
    優點：
    1. 可達99%以上之去除率
    2. 可收集細小之微粒
    3. 微粒可以濕式或乾式收集
    4. 壓力降低及能量需求較低
    5. 除腐蝕、附著問題外，操作正常
    6. 可在高溫下操作
    缺點：
    1. 高建造成本
    2. 需小心維護以免人員受高電壓影響
    3. 易受煙塵負荷及流率之影響
    4. 收集效率會逐漸退化
  • 袋式集塵器
    優點：
    1. 可乾式收集
    2. 可察覺操作性質減低
    3. 可收集小微粒
    4. 可達99%以上之去除率
    缺點：
    1. 高建造成本
    2. 需小心維護以免人員受高電壓影響
    3. 易受煙塵負荷及流率之影響
    4. 收集效率會逐漸退化
